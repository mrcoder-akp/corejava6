Like variable ,array can also be created and used by program to keep data in memory..
1. Array can hold multiple vlaues/data
2. It can hold similar type of values only.
3. An array is data structure for storing more then one data item that has similar data type.
4. An array is collection of similar types of data item which share comman name It is also called linear data structure.
5. Memory allocation of the array will be static..
6. Size of the array is also fixed
7. In array memory is allocated in contiguous manner
8. Accessing data/value from the array is much faster than accessing data/value from linkedlist

Example..

Think of an array like a train with compartments. Each compartment can hold one piece of luggage (value), and the compartments are numbered (index) so you know where to find or place your luggage.

Types of Array

1. One-Dimantional array

2. Two-Dimantional array

3. Multi-Dimantional array


One-Dimantional array

A one-dimensional array is like a single row of elements arranged in a straight line. It is similar to a list where elements are stored one after another in a sequence.
Think of it as a line of boxes, each containing one value.

Example: [10, 20, 30, 40, 50]

Each box has an index (starting from 0) that helps you access the value.

For example:
array[0] = 10
array[2] = 30

when storing a simple list of values like test scores, names, etc.

Two-Dimenstional array

It is array of arrays.
A two-dimensional array is like a grid or a table with rows and columns. It stores data in a matrix form.
Think of it as multiple rows of boxes stacked together, where each row has the same number of columns.

Example...
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]

array[0][0] = 1 (First row, first column)
array[1][2] = 6 (Second row, third column)

import java.util.*;
class Arrayresizble {
    private String[] arr;
    private int i = -1;

    public Arrayresizble() {
        arr = new String[4];
    }

    public void add(String str) {
        if(arr.length==i+1)
        {
        growsize();
        }
         arr[++i] = str;
    }
    public void growsize()
    {
        String[] x=new String[arr.length+arr.length/2];
        for (int c = 0; c <= i; c++) 
        {
            x[c]=arr[c];
        }
        arr=x;
        System.out.println("size increging");
        
    }

    public int capacity() {
        return arr.length;
    }

    public void showString() {
        for (int c = 0; c <= i; c++) {
            System.out.println(arr[c]);
        }
    }
}
