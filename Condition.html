In Java, conditions are expressions or statements that control the flow of a program by evaluating whether something is true or false. They help the program make decisions and execute different blocks of code based on the result.

If: Condition

Executes a block of code if a given condition is true.

public class DiscountFinder {
  public static void main(String[] args) {
    double amount = 12000;
    double discount = 0.0;

    if (amount >= 10000) {
      discount = amount * 20 / 100;
    }
    if (amount >= 7500 && amount < 10000) {
      discount = amount * 15 / 100;
    }
    if (amount >= 5000 && amount < 7500) {
      discount = amount * 10 / 100;
    }
    if (amount >= 2500 && amount < 5000) {
      discount = amount * 5 / 100;
    }
    System.out.println("Purchase amount was: " + amount);
    System.out.println("Discount amount is: " + discount);
    double pa = amount - discount;
    System.out.println("Payable amount is: " + pa);
  }
}

If-else

Executes one block of code if the condition is true and another block if the condition is false.

public class DiscountFinder1 {
  public static void main(String[] args) {
    double amount = 12000;
    double discount = 0.0;

    if (amount >= 10000) {
      discount = amount * 20 / 100;

      if (amount >= 7500) {
        discount = amount * 15 / 100;

        if (amount >= 5000) {
          discount = amount * 10 / 100;
        } else {
          if (amount >= 2500) {
            discount = amount * 5 / 100;

        }

    System.out.println("Discount amount is: " + discount);
    double pa = amount - discount;
    System.out.println("Payable amount is: " + pa);
  }
}

Division Example

public class Division {
  public static void main(String[] args) {
    int hindi = 56, english = 45, math = 60, physics = 50, chemistry = 68;
    int total = hindi + english + math + physics + chemistry;
    double per = (double) total * 100 / 500;
    System.out.println("Total marks: " + total);
    System.out.println("Percentage: " + per);
    if (per >= 60) {
      System.out.println("First division");
    }
    if (per >= 45 && per < 60) {
      System.out.println("Second division");
    }
    if (per >= 33 && per < 45) {
      System.out.println("Third division");
    }
    if (per < 33) {
      System.out.println("Fail");
    }
  }
}

Discount Finder with Scanner

import java.util.Scanner;

public class DiscountFinder5 {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    System.out.print("Enter purchasing amount: ");
    double amount = sc.nextDouble();
    System.out.println("Purchase amount was: " + amount);
    if (amount < 2500) {
      System.out.println("Sorry you are not eligible for any discount");
    } else {
      double discount = 0.0;
      if (amount >= 10000) {
        discount = amount * 20 / 100;
      } else if (amount >= 7500) {
        discount = amount * 15 / 100;
      } else if (amount >= 5000) {
        discount = amount * 10 / 100;
      } else if (amount >= 2500) {
        discount = amount * 5 / 100;
      }
      System.out.println("Discount amount is: " + discount);
      double pa = amount - discount;
      System.out.println("Payable amount is: " + pa);
    }
  }
}

switch Statement:

Used for checking multiple possible values of a single variable.

    switch (day) {
        case 1: System.out.println("Monday"); break;
        case 2: System.out.println("Tuesday"); break;
        default: System.out.println("Invalid day");
    }

Logical operators like &&, ||, and ! are often used in conditions to combine or modify them. Conditions are crucial for controlling program flow and decision-making.
