We can set/define accessibility/visibility of class,methods and variables by using access modifiers.There are three keywords given in java. Inside class we can define variables,methods,classes,etc
These(variables,methods,classes) will be members of the class
These members can be made visible or invisible in other classes by using one of following access modifiers

1.public
2.protected
3.private
4.Default

We will use one of the above keywrod as access modifier
If: public or private or protected will not be used then member will be default

Package is container of classes in java. public void method1() //public { } protected void method2() //protected { } void method3() //default { } private void method4() //private { }
Access Modifiers Same Package Visibility Other Package Visibility
public In all classes of this package In all classes of this package
protected In all classes of this package In child classes of this package
default In all classes of this package Not visible in this package
private Not visible in this package Not visible in this package

Class can be public or default only.It means class can not be private or protected
Nested class/Inner class can be public,proected,default or private

Example...Public Modifiers

p1
package p1;

public class AA 
{
public void show()
{
System.out.println("Welcome");
}
}
package p1;

public class BB 
{
public void method()
{
AA obj=new AA();
obj.show();
}
}
package p1;

public class CC extends AA
{
public void method()
{
show();
}
}
p2

import p1.AA;

Nested-Class

A Nested class in Java is a class defined within another class. This helps to logically group classes that are only used in one place or improve code readability and organization. There are two main types of nested classes:
1.Static Nested Class.
2.Non-static Inner Class.
