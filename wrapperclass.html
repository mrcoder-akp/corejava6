By using Wrapper class we can convert a primitive datatype value into object form.
Wrapper classes in Java are used to convert primitive data types (like int, double, etc.) into objects.
They are part of the java.lang package and provide a way to use primitives as objects when needed.
Each primitive datatype a wrapper class is given.
Used in collections like ArrayList, where primitives cannot be stored directly.

Primitive Datatype Wrapper Class
byte Byte
short Short
int Integer
long Long
float Float
double Double
boolean Boolean
char Character

Super class of these classes are Number Class
Number class is abstract class because this class is not need instantiation only this class inherit child class.
Number class is abstract super class of all classes
Java.lang package contain these classes.

Boxing

Creating an object of wrapper class to keep data into it is called boxing.
Integer num1=new Integer(90);

AutoBoxing

Creating object of wrapper class automatically to keep data into it is called auto-boxing
Integer num1=90;

Auto-Unboxing

Converting object of wrapper class into primitive datatype is called auto-unboxing
Integer num1= new Integer(90); System.out.println(num1); int num2=num1.intValue(); System.out.println(num2); OUTPUT 90

Utility Methods

Wrapper classes provide many utility methods, such as parsing strings into primitives or vice versa.
Wrapper classes provide methods like parseInt(), valueOf(), etc.

String str = "123"; int num = Integer.parseInt(str); // Converts string to int Converting String to Primitive int num = Integer.parseInt("123"); double dbl = Double.parseDouble("45.67"); Converting Primitive to String: String str = Integer.toString(123); String str2 = Double.toString(45.67);
