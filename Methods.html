Following commands can be written inside the body of the method:

  • Command to create variable
  • Command to assign value to the variable
  • Command to perform arithmetic operation
  • Command to perform comparison operation
  • Command to perform logical operation
  • Command to apply condition
  • Command to execute loop
  • Command to return value
  • Command to call method
  • Command to create object
  • Command to create array

Non Static Method

To call a non-static method, you must create an object and use its reference.
Non-static methods belong to an object, not the class.
Can access both static and non-static members
Mostly used for object-related operations.
You cannot call them directly inside main() without an object.

Example..

class NonStatic {
  // Non-static method
 public void nonstatic() {
      System.out.println("Hello from Non-Static Method!");
  }
  public static void main(String[] args) {
    // Creating an object to call the non-static method
      NonStatic obj = new NonStatic();
      obj.nonstatic(); // Calling the non-static method
  }
}
output 
Hello from Non-Static Method!

Static Method

A static method is a method that belongs to the class, not to any specific object.
It can be called without creating an object of the class.
You can call it using the class name directly.
It is declared with the static keyword.
Can only access static variables and other static methods directly.
In simple words, static methods are for class-level operations and are not connect to any object.
❌ It cannot access instance (non-static) variables directly.
A static method can access only static variables directly

Example..

  class Staticmethod {
    // Static method
    static void showMessage() {
          System.out.println("This is a Static Method!");
      }
  
      public static void main(String[] args) {
          // Calling static method using class name
          Staticmethod.showMessage();
      }
  }
output 
This is a Static Method!

Method to method Calling

Method-to-method calling in Java means calling one method from another method within the same class or a different class.

1️⃣Calling a Method from Another Method in the Same Class

We can directly call a method inside another method using its name (if it's in the same class).If the method is non-static, we must use an object.If the method is static, it can be called directly or using the class name.

Example: Calling Methods in the Same Class

class Example 
 {
    void method1() {
    System.out.println("Method 1 called");
    method2(); // Calling method2 from method1
  }

    void method2() {
        System.out.println("Method 2 called");
    }

    public static void main(String[] args) {
        Example obj = new Example();
        obj.method1(); // Calling method1, which calls method2
    }
}
output 
Method 1 called
Method 2 called

2️⃣ Calling a Method from Another Class

We must create an object of the other class before calling its method (if it's non-static). If the method is static, it can be called directly using the class name.

Example: Calling a Methods from Another Class

  class ClassA {
    void display() {
        System.out.println("Method in ClassA");
    }
}

class ClassB {
    public static void main(String[] args) {
        ClassA obj = new ClassA(); // Creating an object of ClassA
        obj.display(); // Calling method from another class
    }
}
output 
Method in ClassA

3️⃣ Calling Static and Non-Static Methods

Static methods can be called directly inside another static method. Non-static methods require an object to be called.

Example: Calling Static and Non-Static Methods

  class Test {
    static void staticMethod() {
        System.out.println("Static Method Called");
    }

    void nonStaticMethod() {
        System.out.println("Non-Static Method Called");
        staticMethod(); // Static method called inside non-static method
    }

    public static void main(String[] args) {
        staticMethod(); // Direct call (since it's static)

        Test obj = new Test();
        obj.nonStaticMethod(); // Calling non-static method
    }
}
output 
Static Method Called
Non-Static Method Called
Static Method Called

Method Chaining

Method chaining happens when a method returns an object and we call another method on that returned object in a single statement.

Example.. Non-Static method Chaining

  class AA {
    public AA show() {  // Returning 'this' for chaining
        System.out.println("Avinash");
        return this;
    }

    public AA display() {  // Another method for chaining
        System.out.println("Pandey");
        return this;
    }
}

class BB {
    public AA disp() {
        return new AA(); // Returning an instance of AA
    }

    public static void main(String[] args) {
        new BB().disp().show().display(); // Now it's method chaining
    }
}
output 
Avinash
Pandey

Example..Static method Chaining

  class AA {
    public static AA show() { // Static method returning class type
        System.out.println("Avinash");
        return new AA(); // Returning new instance of AA
    }

    public static AA display() { // Another static method returning class type
        System.out.println("Pandey");
        return new AA(); // Returning new instance of AA
    }
}

class BB {
    public static AA disp() {
        return new AA(); // Returning an instance of AA
    }

    public static void main(String[] args) {
        disp().show().display(); // Static method chaining
    }
}
output 
Avinash 
Pandey

Static methods belong to the class, not an instance, so this cannot be used inside a static method.

Example... method chaining with both static and non-static methods

1️⃣Static methods return a new instance of the class (since this cannot be used in static methods).
2️⃣ Non-static methods return this (to maintain the same object for chaining).

  class AA {
    public static AA create() { // Static method returning a new instance
        System.out.println("Static: Creating instance");
        return new AA(); 
    }

    public AA show() { // Non-static method returning the same object
        System.out.println("Instance: Avinash");
        return this; // Returning 'this' for method chaining
    }

    public AA display() { // Non-static method returning the same object
        System.out.println("Instance: Hello");
        return this; // Returning 'this' for method chaining
    }
}

class BB {
    public static void main(String[] args) {
        AA.create().show().display(); // Chaining static & non-static methods
        or 
        new AA().show().display(); // Chaining static & non-static methods
    }
}
output 
Static: Creating instance
Instance: Avinash
Instance: Hello

Difference Between Method Calling and Method Chaining

Feature Method Calling 🚀 Method Chaining 🔗
Definition Calling a method individually Calling multiple methods sequentially using returned objects
Return Type Can return any value (void, int, String, etc.) Must return this (or another object of the same type)
Object Requirement Can be called on any object Must return an object to continue chaining
Code Style Regular method calls (separate statements) Compact and readable, combines multiple calls
Flexibility More flexible, can return different types Limited to methods that return an object
Use Case When methods work independently When methods modify and return the same object
