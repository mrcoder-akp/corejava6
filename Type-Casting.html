It is one of the way to change type of any value
Typecasting in Java means converting one data type into another. It is helpful when you need to change the type of a value to make it compatible with operations or variables.

=>(type)value

=>(type)variable

There are two types of typecasting:

1) Explicit typecasting (Narrowing): This type converts big to little.
You manually convert a larger data type into a smaller one.
There is a risk of losing data during conversion.
Syntax: (type)value

2) Implicit typecasting (Type-Permission): This type casting converts little to big.

Example of Explicit typecasting:

int x;
x=(int)56.78; //Here explicit

Example of Implicit typecasting:

float x;
x=78; //Here implicit

Example 1:

public class TypeCasting
{
  public static void main(String[] args)
  {
    int num;
    num=(int)90.99;
    System.out.println(num); //Calling println() method
  }
}
----------
output
90
-----------

Example 2:

Explicit Typecasting: Requires manual effort, may lose data.
Implicit Typecasting: Done automatically by Java, safe.
Works with primitive types (int, float, char, etc.) and objects with appropriate conversions.
