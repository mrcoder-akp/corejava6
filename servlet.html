By using this technology we will build server side programe
These programs will be process to http request and genrate http response
Life cycle of these programs will be managed by web server,web server will be container of these program These program will be called servlet.
These programs will be responsible to perform following taks..
1.Accepting HTTP request and data
2.Generating HTTP response
3.Sending response to the web browser
Inorder to build this program we will be user classes and interface provided be web server Vendors.So jar file of these classes and interfaces must be added into our projects.
If you are using tomcat web server then you can get this jar files from the lib folder of tomcat.Name of this har file is (servlet-api.jar)

Javax.Servlet. Servlet interface

Every servlet(server side program that you are building using servlet technology) must impliments this interface either directly or indirectly

Servlet interface has following %5% method

1.public void init(servletConfig config)
2.public void service(servletRequest req,servletresponse res)
3.public void destroy()
4.public String getservletInfo()
5.public servletConfig getservletCongif()
first three method of servlet interface are Life cycle method of servlets. these methode will be called by web server

Service method

This method will be called by web server each time web server will get http request for the servlets.
web service pass by two object to the service method
1.Object of HttpservletRequest Interface
2.Object of HttpservletResponse Interface
Service can call method of these interface....

Request object

It is an object of HttpServletRequest interface.web server create this object before calling service method of request Servlet of jsp page. Web server passesd reference of these object to the service method.web server keep input into this object.Request object keeps data into key value pair.

Response Object

It is an Object of httpServletresponse interface.Service method can call this method of Httpservletresponse interface to the send response to the web browser.

Life cycle of servlets

Life cycle of servlet is managed by web server
It has following 5 phases
1.Loading servlets:Web server loads class of the servlets
2.Instantiating servlets:Web server creates an object from class of the servlets
3.Initializing servlets:Web server initializes that object.Web server calls init() life cycle method here
4.Prcoessing HTTP request:Web server calls service method to prcoess http request.Web server calls this method each time web server recieves HTTP request from browser.In this method web server passes following two objects as an argument.
a.request object
b.response object
5.Destroying servlets:Web server destroys servlets when web server stops or restarts
