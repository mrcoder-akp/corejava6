Java is a programming language and a platform. Java is an object-oriented programming language (OOPS) and a secure programming language. Java is a platform-independent language. It means Java works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.). Java was developed by Sun Microsystems (which is now the subsidiary of Oracle) in the year 1995.James Gosling is known as the father of Java. The original name of the Java programming language was Oak, after the oak tree outside James Gosling's office. However, the name was changed to Java after a trademark search revealed that Oak was already used by Oak Technologies.
"The Java project was started in 1991 by Gosling, Mike Sheridan, and Patrick Naughton at Sun Microsystems. The language was designed for interactive television but was too advanced for the digital cable television industry at the time. The first public implementation of Java was released as Java 1.0 in 1996. The name "Java" was inspired by a type of coffee grown on the Indonesian island of Java. Other proposed names for the language included DNA, Silk, WebDancer, Pepper, Neon, WRL, Jolt, WebSpinner, and Impact".

Platform

Any hardware or software environment in which a program runs, is known as a platform. Since Java has a runtime environment (JRE) and API, it is called a platform.
