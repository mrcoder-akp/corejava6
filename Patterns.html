Nested-Loop

Nested loop is required to make above pattern.
Outer loop will be for row and inner loop will be for column.
Total rows are n.
Rowwise columns are row number.

A nested loop in Java is a loop inside another loop. The inner loop executes completely for each iteration of the outer loop. This is useful when working with multi-dimensional data or patterns that require repetitive operations within repetitive operations.

Outer Loop: The main loop that controls the overall execution.

Outer Loop:Inner Loop: The loop that gets executed fully for every iteration of the outer loop.

Outer Loop:Execution: The inner loop runs completely for each iteration of the outer loop.

Example..

for (initialization; condition; increment/decrement) { // Outer Loop
   for (initialization; condition; increment/decrement) { // Inner Loop
         // Code to execute inside the inner loop
    }
        // Code to execute inside the outer loop
}
 
 public class NestedLoopExample {
 public static void main(String[] args) {
  for (int i = 1; i <= 5; i++) { // Outer loop for rows
    for (int j = 1; j <= 5; j++) { // Inner loop for columns
      System.out.print(i * j + "\t"); // Multiplying and printing
    }
    System.out.println(); // New line after each row
    }
    }
    }

                public class App3

                 public static void main(String[] args) 
                 {
                   int n = 9; // Size of the pattern
                    for (int r = 0; r < n; r++) 
                    {
                    for (int j = 0; j < n; j++) 
                     {
                      if (r == j || r + j == n - 1)
                       {
                       System.out.print("*");
                       } 
                       else 
                        {
                          System.out.print(" ");
                        }
                        }
                        System.out.println();
