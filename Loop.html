It makes coding simple and easy.
You write a command once, and that command will run n times.

n can be finite or infinite.
You must define a loop using one of the following keywords:

  • For-loop
  • While
  • do-While
    •

These keywords are known as loop constructs---

Using for: When n is finite and fixed.

Using while: When n is finite or infinite and not fixed.

Using do-while: When n is finite or infinite and not fixed.

For-Loop

When to Use: When n is finite and fixed.

public class App {
  public static void main(String[] args) {
    for(int ctr = 1; ctr <= 10; ctr++) {
      System.out.println(ctr);
    }
  }
}

public class App2 {
  public static void main(String[] args) {
    for(int ctr = 10; ctr >= 1; ctr--) {
      System.out.println(ctr);
    }
  }
}

public class ForDemo {

While-Loop

When to Use: When n is finite or infinite and not fixed. The condition is checked before execution.

    public class WhileDemo {
      public static void main(String[] args) {
        int ctr = 0;
        while(ctr < 10) {
          System.out.println("jai shree ram");
          ctr++;
        }
      }
    }

Do-While Loop

When to Use: When n is finite or infinite and not fixed. The statement is executed at least once before the condition is checked.

    public class DoWhileDemo {
      public static void main(String[] args) {
        int ctr = 0;
        do {
          System.out.println("I LOVE MY INDIA");
          ctr++;
        } while(ctr < 10);
      }
    }
