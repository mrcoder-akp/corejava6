http://localhost:8080/banking-app/login/login.jsp

This is url of the page(login.jsp) Url means uniform resource locator Every page(Static or dynamic) will have an unique URL
Every Url has following four parts.
Protocol-->http
IP address of server-->localhost
Port number of server(Tomcat)-->8080
Resource name-->banking-app/login/login.jsp

Web browser will send http request to the web server and web server send response(Any page) to the web brower.
To build front end of this application we will use HTML,CSS and Java Script
To make it dynamic we will use following two technologies of java
1.Servlets technology
2.JSP technology
To manage data in database we will use JDBC
