Our program will perform arithmetic operations, comparison operations, as well as logical operations on the data kept in variables.
Operators are used to write commands to perform these operations.
Every operator returns a value after performing the operation.

Using Operators in Commands

We can use the following operators in our command to perform arithmetic, comparison, and logical operations:

  • +,-,*,/,% // Arithmetic operators
  • +=,-=,*=,/=,%= // Arithmetic assignment operators
  • ++,-- // Increment and decrement operators
  • <, <=, >, >=, ==, != // Comparison operators
  • && (AND), || (OR), ^ (XOR), ! (NOT) // Logical operators

Arithmetic Operators

Operators like +, -, *, /, and % are used for arithmetic operations. The + operator performs addition and returns an int value.

    Example:
    public class ArithmeticOperator {
        public static void main(String[] args) {
            int x = 100, y = 16;
            int z = x % y;
            System.out.println(z);//4
        }
    }
        
    public class ArithmeticOperator1 {
        public static void main(String[] args) {
            double tempf = 101.0;
            System.out.print("Temperature in Fahrenheit is: ");
            System.out.println(tempf);
            // Code to convert temperature into Celsius
            double tempc = (tempf - 32) * 5 / 9;
            System.out.print("Temperature in Celsius is: ");
            System.out.println(tempc);
        }
    }
    -------------
    output 
    Temperature in Fahrenheit is: 101.0
    Temperature in Celsius is: 38.333333333333336
    --------------
        
    public class ArithmeticOperator2 {
        public static void main(String[] args) {
            String x = "10", y = "20";
            String z = x + y;
            System.out.println(z);//1020
        }
    }
        
    public class ArithmeticAssignmentOperator {
        public static void main(String[] abc) {
            int salary = 70000;
            System.out.println("Salary before increment: " + salary);
            salary += 9000;
            System.out.println("Salary after increment: " + salary);
        }
    }
    ------------------
    output 
    Salary before increment: 70000
    Salary after increment: 79000
    --------------------

Note: Every operator returns a value after performing the operation.

Increment Operators

    public class IncrementOperator {
        public static void main(String[] abc) {
            int x = 10;
            // Increment x by 1 using various methods
          //  x = x + 1;  // Good
           // x += 1;     // Better
            ++x;        // Best
            System.out.println(x);//11
        }
    }
        
    public class IncrementOperator3 {
        public static void main(String[] abc) {
            int x = 10, y;
            y = ++x; // Value of x will be incremented by 1 and assigned to y
            System.out.println("x = " + x);//11
            System.out.println("y = " + y);//11
        }
    }
        
    public class IncrementOperator5 {
        public static void main(String[] abc) {
            int x = 10, y;
            y = x++ - ++x;
            System.out.println("x = " + x); //12
            System.out.println("y = " + y); // -2
        }
    }

Comparison Operators

Operators like <, <=, >, >=, ==, != are used to compare numeric values. These operators return a boolean value (true or false).

    int x = 10, y = 20;
    boolean z = x < y;
    System.out.println(z);//true

Logical Operators

Operators like &&, ||, and ^ are used with boolean values and return a boolean value:
These operators are used with boolean value and return boolean value
Accepts two boolean value and returns single boolean value

  • && Returns true if both conditions are true.
  • || Returns true if at least one condition is true.
  • ^ Returns true if only one condition is true.
    public class LogicalOperator {
        public static void main(String[] args) {
            boolean r1 = true && true;
            boolean r2 = true || true;
            boolean r3 = true ^ true;
            System.out.println(r1);//true
            System.out.println(r2);//true
            System.out.println(r3);//false
        }
    }
        
    public class LogicalOperator5 {
        public static void main(String[] args) {
            int x = 0, y = 0;
            for (int c = 0; c < 10; c++) {
                boolean z = ++x < 5 && ++y < 10;
            }
            System.out.println(x); // 10
            System.out.println(y); // 4
        }
    }
        
    public class LogicalOperator7 {
        public static void main(String[] args) {
            int x = 0, y = 0;
            for (int c = 0; c < 10; c++) {
                boolean z = ++x < 5 ^ ++y < 10;
            }
            System.out.println(x); // 10
            System.out.println(y); // 10
        }
    }
        
            class LogicalOperator8 {
                public static void main(String[] args) {
                    int x = 0, y = 0;
                    for (int c = 0; c < 10; c++) {
                        boolean z = ++x < 5 || ++y < 10;
                    }
                    System.out.println(x); // 10
                    System.out.println(y); // 6
                }
            }

Note

If first condition returns false then && returns false without executing second condition
If first condition returns true then || returns true without executing second condition

  • =: Assigns a value to a variable.
  • +=: Adds the value to the variable and assigns it.
  • -=: Subtracts the value from the variable and assigns it.
  • *=: Multiplies the value with the variable and assigns it.
  • /=: Divides the value of the variable and assigns it.
  • <: Less than
  • >: Greater than

  • >=: Greater than or equal to
  • ==: Equal to
  • !=: Not equal to
  • && (Logical AND): True if both conditions are true.
  • || (Logical OR): True if at least one condition is true.
  • ^ (Logical XOR): True if only one condition is true.
  • ! (Logical NOT): Reverses the boolean value.

  • ~: Bitwise complement
  • <<: Left shift
  • >>: Right shift

Ternary Operator

The ternary operator in Java is a shorthand for the if-else statement. It evaluates a condition and returns one of two values depending on whether the condition is true or false.

  • condition: A boolean expression that evaluates to true or false.
  • value1: The value returned if the condition is true.
  • value2: The value returned if the condition is false.
int a = 10, b = 20;
int max = (a > b) ? a : b; // Returns the larger value
System.out.println("Max: " + max); // Output: Max: 20

Explanation:

  • (a > b) is the condition.
  • If a > b is true, the result is a.
  • If a > b is false, the result is b.
