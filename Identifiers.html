Identifiers

An identifier is the name of a class, variable, or method. It must follow specific naming rules.

Naming Rules

  • Name must begin with an alphabet or $ or _(underscore).
  • Space is not allowed in a name.
  • Keywords cannot be used as names.

Naming Conventions

Following conventions should be followed in naming inorder to improve readbility of code
Name should be meaningful

  • Class names should be in TitleCase.
  • Method, variable, and argument names should be in camelCase.
  • Example....
  • MyNameAvinashPandey
  • myNameIsAvinashPandey
  • mynameisavinashpandey
  • MYNAMEISAVINASHPANDEY

Rules for Identifiers:

Must start with a letter (A-Z or a-z), a dollar sign ($), or an underscore (_).

Cannot start with a digit (0-9).

Can contain letters, digits, dollar signs, and underscores but no spaces or special characters like @, #, &, etc.

Cannot use Java keywords as identifiers (e.g., class, if, else).

Case-sensitive: myVariable and MyVariable are treated as different identifiers.
